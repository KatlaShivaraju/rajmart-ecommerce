# RajMart Ecommerce 🛒

RajMart is a full-stack ecommerce web application built using Spring Boot and MySQL. It provides secure authentication, product management, shopping cart, and order processing features.

## Features
- User Registration & Login
- Product Management
- Add to Cart
- Order Placement
- Secure Authentication
- Admin Dashboard

## Tech Stack
- Java
- Spring Boot
- Spring Security
- MySQL
- Maven
- HTML/CSS/JavaScript

## Installation
1. Clone repository
2. Configure MySQL database
3. Run:
```bash
mvn spring-boot:run
